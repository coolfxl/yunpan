# yunpan
个人云盘（html5+css3+js）
下面是项目截图，个人模仿腾讯微云实现的一个小demo，利用了es6的语法，暂时没有做后台

主要实现功能：新建文件夹，文件重命名，文件删除，文件移动，文件单选全选，文件分享
